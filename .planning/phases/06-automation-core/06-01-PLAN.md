# Plan 06-01: Command History

**Phase:** 6 - Automation Core
**Requirement:** AUTO-01 - 系统保存命令执行历史

## Objective

实现命令历史记录功能，自动保存终端执行的命令，提供查询、搜索和重新执行能力。

## Scope

### In Scope
- 命令历史数据库表和模型
- 后端命令历史 API (记录、查询、删除、清空)
- 前端命令历史面板组件
- 命令搜索和过滤
- 重新执行历史命令

### Out of Scope
- 脚本库管理 (PLAN-02)
- 文件监控 (PLAN-03)
- API 调用 (PLAN-04)

## Tasks

### Task 1: 数据库 Schema 扩展

**File:** `natsu/src-tauri/src/db/schema.rs`

添加命令历史表：

```sql
CREATE TABLE IF NOT EXISTS command_history (
    id TEXT PRIMARY KEY,
    command TEXT NOT NULL,
    working_directory TEXT,
    exit_code INTEGER,
    duration_ms INTEGER,
    executed_at INTEGER NOT NULL,
    session_id TEXT
);

CREATE INDEX IF NOT EXISTS idx_command_history_time ON command_history(executed_at DESC);
CREATE INDEX IF NOT EXISTS idx_command_history_command ON command_history(command);
```

pub struct CommandHistoryQuery {
