```sql
CREATE TABLE IF NOT EXISTS file_watchers (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    path TEXT NOT NULL,
    recursive INTEGER DEFAULT 1,
    event_types TEXT NOT NULL,
    enabled INTEGER DEFAULT 1,
    trigger_script_id TEXT,
    created_at INTEGER NOT NULL,
    FOREIGN KEY (trigger_script_id) REFERENCES scripts(id)
);

CREATE TABLE IF NOT EXISTS file_events (
    id TEXT PRIMARY KEY,
    watcher_id TEXT NOT NULL,
    event_type TEXT NOT NULL,
    path TEXT NOT NULL,
    details TEXT,
    timestamp INTEGER NOT NULL,
    FOREIGN KEY (watcher_id) REFERENCES file_watchers(id)
);

CREATE INDEX IF NOT EXISTS idx_file_events_time ON file_events(timestamp DESC);
```

### Task 3: 文件监控服务

**File:** `natsu/src-tauri/src/file_watcher.rs` (新建)

```rust
use notify::{Watcher, RecursiveMode, Event, EventKind};
use std::sync::mpsc::channel;
use tauri::{AppHandle, Emitter};

pub struct FileWatcherService {
    watchers: HashMap<String, Box<dyn Watcher>>,
}

impl FileWatcherService {
    pub fn new() -> Self;
    
    pub fn start_watcher(
        &mut self,
        id: String,
        path: String,
        recursive: bool,
        event_types: Vec<String>,
        app_handle: AppHandle,
    ) -> Result<(), String>;
    
    pub fn stop_watcher(&mut self, id: &str) -> Result<(), String>;
    
    pub fn stop_all(&mut self);
}
```

事件流程：
1. 文件变更 → notify 触发事件
2. 记录到 file_events 表
3. 通过 Tauri event 发送到前端
4. 如果配置了 trigger_script_id，执行对应脚本

## Event Types

| Event | Description |
|-------|-------------|
| `create` | 文件或目录创建 |
| `modify` | 文件内容修改 |
| `delete` | 文件或目录删除 |
| `rename` | 文件重命名 |
| `any` | 任何变更 |
