## Layout Contract

### Desktop/Web (width >= 768px)

Three-column layout with collapsible sidebars:

```
+--------+------------------+----------+
|        |                  |          |
| Sidebar|   Main Editor    | Preview  |
| 240px  |   (flexible)     | 320px    |
|        |   / Chat Area    |          |
|        |                  |          |
+--------+------------------+----------+
```

**Sidebar (left, 240px):**
- File tree navigation
- Search input
- Settings entry point
- Collapsible (icon-only mode at 48px)

**Main Editor (center, flexible):**
- Markdown editor with real-time rendering
- AI chat interface (Phase 4+)
- Active content area

**Preview Panel (right, 320px):**
- Note preview / backlinks
- File list / recent items
- Collapsible (hidden mode)

### Mobile (width < 768px)

Chat-style layout with side drawer:

```
+------------------+
|  [Menu]  Title   |
+------------------+
|                  |
|   Main Editor    |
|   / Chat Area    |
|   (full width)   |
|                  |
+------------------+
```

**Side Drawer (slide-in from left):**
- File tree
- Navigation
- Settings
- Triggered by hamburger menu icon

## Wiki-Link Interaction Contract

From CONTEXT.md decisions D-09 to D-13:

| Interaction | Behavior |
|-------------|----------|
| Type `[[` | Show autocomplete with existing notes |
| Autocomplete matching | Fuzzy match on note titles |
| Select suggestion | Insert `[[note-name]]` at cursor |
| Click existing link | Navigate to target note |
| Broken link (target not found) | Display with warning style, show "Create?" tooltip on hover |
| Case sensitivity | Default: sensitive; toggle in settings for insensitive |

**Link styling:**
- Valid link: Accent color, underline on hover
- Broken link: Destructive color with dashed underline
- Hover: Show preview card (Phase 2+)

## Accessibility Requirements

- All interactive elements keyboard accessible
- Focus indicators visible (accent color ring)
- Color contrast: WCAG AA minimum (4.5:1 for body text)
- Touch targets: minimum 44x44px on mobile
- Screen reader: ARIA labels on icon-only buttons
- Reduced motion: respect `prefers-reduced-motion`
