## Spacing Scale

Declared values (multiples of 4px, matching existing patterns):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, tight list spacing |
| sm | 8px | Compact element spacing, button padding |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, panel gaps |
| xl | 32px | Layout gaps between major panels |

Exceptions:
- Touch targets on mobile: minimum 44px (for accessibility)
- Tab trigger height: 40px (h-10)
- Card header padding: p-6 (24px)
- Card content padding: p-6 pt-0 (24px with no top)

    />

  {/* Preview */}
  {nextRuns.length > 0 && (
    <div>
      <Label>下次执行时间</Label>
      <div className="text-sm text-muted-foreground mt-1">
        {nextRuns.slice(0, 3).map((time, i) => (

      </div>
    </div>
  )}
</div>
